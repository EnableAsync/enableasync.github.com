2. 创建集群

这里需要注意的点有：

  1. 不要设置集群 name，在我本地，如果设置了 name 会导致 kubeconfig 无法导出。
  2. 要使用 sudo，在我本地，如果不使用 sudo 会导致无法创建集群，原因未知。

如果出现以上信息表示创建成功，可以进行下一步。

3. 导出 kubeconfig
